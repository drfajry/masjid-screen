# البدء بالتجربة الفعلية — خطوات سريعة

تقدر تبدأ التجربة **الآن** على أي جهاز ويندوز (حتى قبل وصول الـ Mini PC).

## المتطلّبات (مرة واحدة)
1. ثبّت **Node.js 18 أو أحدث** من nodejs.org.
2. اتصال إنترنت **عند أول تثبيت فقط** (لبناء المكتبات). بعدها يعمل دون إنترنت.

## التشغيل
من داخل مجلد المشروع:

```bash
npm install      # تنزيل المكتبات فقط (بدون أي بناء C++) — يحتاج إنترنت مرة واحدة
npm start        # تفتح شاشة العرض بملء الشاشة، ويبدأ خادم لوحة التحكم
```

- `Esc` للخروج من ملء الشاشة، `F11` للعودة إليه.

## لوحة التحكم من الجوال
1. وصّل الجوال بنفس شبكة الواي‑فاي.
2. اعرف عنوان IP للجهاز (يظهر في نافذة التشغيل عند الإقلاع).
3. افتح في متصفح الجوال:
   ```
   http://<عنوان-IP>:3777/
   ```
4. الدخول: `admin` / `admin` (غيّر كلمة المرور من تبويب «المستخدمون»).

## ماذا تقدر تجرّب فوراً (بدون أي عتاد إضافي)
- أوقات الصلاة والإقامة، والعدّاد، وشاشة الإقامة، والأذكار.
- تبديل الثيمات (ليلي/الجفالي/زيتوني/فاتح) ورفع شعار الجامع.
- التحكم الحي: بدء الإقامة، إظهار الأذكار، العودة.
- مزامنة الوقت (تلقائي/يدوي)، الجنازة، المناسبات، السجل، المستخدمون.
- مراقبة المكسر عبر الشبكة: جرّبها بإرسال طلب تجريبي:
  ```bash
  curl -X POST http://<IP>:3777/api/signal -H "Content-Type: application/json" -d "{\"active\":true}"
  ```

## يحتاج عتاد فعلي (اختياري — النظام يعمل بدونه)
- **GPS:** جهاز USB‑GPS لضبط الموقع والوقت تلقائياً.
- **مراقبة المكسر:** لوحة USB دخل رقمي، أو متحكّم صغير (Arduino/ESP) يرسل الحالة عبر الشبكة.
لتفعيل قراءة المنفذ التسلسلي ثبّت مكتبتها:
```bash
npm install serialport @serialport/parser-readline
npx electron-rebuild -f -w serialport
```

## بناء نسخة تثبيت للتوزيع (لاحقاً)
```bash
npm run dist     # ينتج مثبّت ويندوز بأيقونة الجفالي داخل مجلد dist/
```
للتحديثات التلقائية: اضبط مستودعك في `build.publish` ثم `npm run dist -- --publish always` مع `GH_TOKEN`.

## ملاحظة
- وضع المتصفح فقط `npm run server` مخصّص للاختبار السريع عبر متصفح بدل Electron.
- التجربة الكاملة الموصى بها هي `npm start` (وضع Electron).
- قاعدة البيانات تخزين JSON (ملف `data/masjid.json`) — لا تحتاج أي أدوات بناء.
